# ASG-WDS-Monitor-Early-Warning-System
本项目设计了一套基于"空天地水"一体化协同的水质水坝安全监测与预警系统，采用"无人机+无人船"协同作业模式，通过三维激光雷达对大坝进行高精度建模，结合多种水质传感器对水域进行多参数监测，实现对水坝结构安全和水质污染的全方位、实时监测与智能预警。系统通过ESP32+华为云平台实现数据采集与处理，并利用树莓派5+Lora模块构建地面站进行实时监控，同时基于Airsim+ROS+QGC构建仿真环境，为系统测试与优化提供支持。该系统将物联网、云计算、边缘计算和人工智能技术有机结合，实现了对水环境和水工设施的智能化监测与预警。
